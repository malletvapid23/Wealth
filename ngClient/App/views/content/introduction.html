Introduction

Can we distribute our monetary resources to organizations by measuring their benefits to the society?

Wealth Economy is an attempt to create a collaborative, customizable system to allow the society manage its monetary resources. It aims to improve the system by empowering the organizations that are more beneficial to the society, using the following steps;

  1. CMRP (Crowd Managed Resource Pool): Creates a pool that holds certain monetary amount to be distributed.
  2. Indexes: Defines various parameters, categories for the organizations that seek benefit to the society.
  3. Ratings: Allows either citizens' ratings or previously measured values to be entered on these 'Indexes' for each organization.
  4. Distribution: Calculates organizations' overall performance on these 'Indexes' and distribute 'CMRP Amount' to the organizations based on the outcome.

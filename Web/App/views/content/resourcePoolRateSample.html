CMRP Rate

CMRP Rate determines the size of the pool, if it gets bigger, the amount that we can control through the pool gets bigger as well. In the previous samples, this number was fixed and it was 10% of the total spendings in the system. However, like other variables in the system, this rate can also be decided collectively.

  • By using + button increase Number of Sales, since CMRP Rate is 0%, initially Index Income will be 0 for all organizations.
  • Increase CMRP Rate by using + next to it, which will increase CMRP Amount to be distributed.
  • Toggle Display between Only My Ratings and All Ratings to see only your ratings or the average of all users'.

CMRP Rate

To be able to make a difference in the system, we're allocating the money that has been in the pool. In the previous samples, this amount was always fixed and it was 50% higher than Total Sales Price. In reality, this money has to come from somewhere.

Ideal way of creating a Resource Pool would be to collect some portion of the money from every transaction, like value added tax. However, as seen already, with two important exceptions;

  1. We the people will control how this pool will be distributed
  2. Every organization in the system can get a share from this pie based on their performance

As like other parameters of the system, this rate could also be decided collectively.

  • By using + button increase Number of Sales, since CMRP Rate is 0%, initially Index Income will be 0 for all organizations.
  • Increase CMRP Rate by using + next to it.
  • When CMRP Rate is getting increased, CMRP Amount that represents the money that we're going to distribute will also be getting increased.
  • Switch Display to All Ratings to see the average of all users' ratings.

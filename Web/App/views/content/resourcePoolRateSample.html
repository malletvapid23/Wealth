CMRP Amount: The Fountain

CMRP Rate determines the size of the pool, if it gets bigger, the amount that we can control through the pool gets bigger as well. In the previous samples, in order to make the difference apparent, this number was fixed to 10% of the total spendings in the system.

At this stage, the application will be used as a simulation tool; we will be creating samples with various cases to show how we could improve the system and allow you to create your own. So, we can easily experiment with the size of the pool as well.

After gathering enough data, if the results convince us that a certain pool can make a difference, then the next phase will be to actually creating the pool. How these resource pools and amounts will be realized may vary. Initially, certain amount can be collected from the volunteers. In the long run, if the number of convinced people will be getting higher in a certain country and there is a political power, to make it much more systematic, it would be ideal to implement such system into the tax system of that country.

In any case, even if the amount that will be distributed through this system would be very small, publicly affecting the market, rewarding the beneficial organizations, would have an important, positive effect to the society.

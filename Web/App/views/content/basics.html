Basics

In the following sample, there are two organizations that are equal to each other in every aspect, except they use different strategies to motivate their employees. In Existing Model, since this difference is ignored, each organization will have the same Total Income.

New Model will introduce Employee Satisfaction index. By using this index, we can enter which organization's employees satisfied with their organization's strategy and based on this new data, organizations' Total Income will now differ.

Difference between the models

In New Model, Total Income is 50 percent higher than Existing Model. This difference will be explained in further chapters.

  • In Existing Model, increase organizations' Number of Sales by clicking + button.
    This will increase Number of Sales in both models to be able compare the results easily.
  • In New Model, update Employee Satisfaction ratings of the organizations by using + and - buttons next to them and see the difference in Total Income fields in both models.

If we could measure Employee Satisfaction of the organizations in real life, we could reward the ones with a higher rating as in this sample. Adding indexes to the system sets additional priorities for the organizations, instead of just focusing to profit.

Although only four different indexes will be presented in the following chapters, as long as they are measurable, more indexes can be added into the system.
